Настройки

API ключи и параметры генерации

Wildberries API
Токен должен иметь доступ к категории «Вопросы и отзывы» с правами Чтение и запись. Создать: ЛК продавца → Интеграции по API → Создать токен.
Perplexity API
Ключ получить: perplexity.ai → Settings → API → Generate. Используется модель sonar для генерации ответов. Оплата по кредитам, ~1-2$ за 1000 коротких ответов.
Системный промпт
Инструкция для AI — что и как отвечать на отзывы. Переменные: {stars} — оценка, {product} — товар, {text} — текст отзыва, {pros} — плюсы, {cons} — минусы

Подписи брендов
Настройте подписи для каждого бренда. Подпись автоматически добавляется в конец ответа, если название бренда есть в наименовании товара отзыва.
Как это работает: при загрузке отзывов сервис автоматически запрашивает бренд каждого товара через Content API WB (по nmId артикулу). Введите точное название бренда (как в карточке товара на WB) и текст подписи. Например, бренд «SNOWPRO» → подпись «С уважением, команда SNOWPRO ❄️». Подпись добавляется только к ответам на товары именно этого бренда.

⚠️ Для работы Content API WB API токен должен иметь доступ к категории «Контент».
